# دليل إعادة بناء قاعدة البيانات

## الخطوات المطلوبة:

### 1. تنفيذ سكريبت إنشاء قاعدة البيانات
قم بتنفيذ الملف `database-schema.sql` في Supabase SQL Editor:
1. افتح Supabase Dashboard
2. اذهب إلى SQL Editor
3. انسخ محتوى ملف `database-schema.sql`
4. نفذ السكريبت

### 2. التغييرات الرئيسية:

#### جدول الوحدات السكنية (Units):
- **رقم الوحدة (unit_number)**: حقل نصي إلزامي - يتم إدخاله يدوياً
- **المساحة (type)**: يتم اختيارها من جدول `unit_types` القابل للتخصيص
- **السعر (price)**: حقل رقمي إلزامي - يتم إدخاله يدوياً من قبل المدير

#### جدول أنواع المساحات (Unit Types):
- قابل للتخصيص بالكامل من قسم التخصيص
- المساحات الافتراضية:
  - 100 متر مربع
  - 150 متر مربع
  - 200 متر مربع
  - 250 متر مربع
  - 300 متر مربع

### 3. المشاريع:
- يمكن عرض جميع المشاريع
- يمكن اختيار مشروع محدد لعرض:
  - الوحدات السكنية الخاصة به
  - الحجوزات الخاصة به
  - العمليات المالية الخاصة به

### 4. صلاحيات المدير:
- إضافة/تعديل/حذف المساحات من قسم التخصيص
- إدخال رقم الوحدة يدوياً
- اختيار المساحة من القائمة
- إدخال سعر الوحدة يدوياً

### 5. الهيكل المبسط:
```
المشروع
  ├─ الوحدات السكنية
  │   ├─ رقم الوحدة (إدخال يدوي)
  │   ├─ المساحة (اختيار من القائمة)
  │   └─ السعر (إدخال يدوي)
  ├─ الحجوزات
  └─ المدفوعات
```

## ملاحظات:
- جميع البيانات القديمة سيتم حذفها
- يجب تنفيذ السكريبت مرة واحدة فقط
- سيتم إنشاء مستخدم افتراضي عند أول تسجيل دخول
